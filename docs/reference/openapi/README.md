# OpenAPI Reference Artifacts

This directory contains raw Aptitude Server API artifacts and Postman exports.

## What These Files Are

- server-side reference material
- useful for transport and contract inspection
- suitable as supporting evidence when changing the `registry/` boundary

## What These Files Are Not

- the canonical source of aptitude behavior
- a substitute for `docs/architecture/*`
- proof that the resolver should mirror server ordering or server-side decisions
